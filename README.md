# Сервер сайд для личного кабинета ЛК Энегргосбыт
## Установка для разработки
Клонируем репозиторий
```
git clone https://github.com/tugaserya/LK-company.git
cd LK-company
```
Установка зависимостей
```
npm i
```
## Запуск
Для разработки
```
npm run dev
```
Обычный запуск
```
npm start
```
